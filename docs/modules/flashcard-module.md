# Flashcard Module 

## Overview
The Flashcard Module is designed to help users create, manage, and review flashcards for effective learning. It provides a user-friendly interface for creating flashcards, organizing them into decks, and tracking progress over time.

## Features 
### Multiple content types 
The Flashcard Module supports various content types for both the front and back of flashcards, including:
- Text
- Images
- Audio
- Video 

Addtionally, the text content type can be enriched with an example pronunciation and transcription. This is however only available for import at the moment.  

### Deck management
Users can create multiple decks to organize their flashcards by subject, topic, or any other categorization method. Each deck can contain an unlimited number of flashcards. Users can easily add, edit, and delete flashcards within each deck.

### Flashcard preview 
When creating or editing a flashcard, users can preview how the flashcard will appear during review sessions. This allows them to make adjustments to the content and formatting as needed.

## Navigation
The module consists of several pages: 
- View all flashcard decks page
- View flashcard deck page
- Edit flashcard deck page
