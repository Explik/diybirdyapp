# Road map 

## 0.0.0 - All the flashcards 
This release focuses on flashcard-based learning. 

### Features
- [x] Create/edit/delete flashcard decks 
- [x] Create/edit/delete flashcards in a deck
- [x] Support for content types, text, image, audio, video flashcards
- [x] Support for concept types, pronounciation, romanization, translation, ...
- [x] Import flashcard tool (from text, CSV, ANKI, ZIP)
- [x] Create flashcard deck ZIP python package
- [x] Manage language tool 
- [x] Manage UI language tool
- [x] Dynamic flashcard-based exercises 
    - Select flashcards exercise
    - Review flashcards exercise (Again, Hard, Good, Easy)
    - Review flashcards exercise (sorting)
    - Write flashcards exercise
    - Tap pairs exercise
- [x] Dynamic pronunciation-based exercises
    - Pronounce text exercise
    - Write transcribe audio exercise
    - Select transcription exercise
- [x] "Select flashcards" session 
- [x] "Review flashcards" session
- [x] "Write flashcards" session
- [x] "Learn flashcards" session
- [x] View session history and statistics
- [x] User authentication (sign up, log in, log out)
- [x] Docker deployment

## 0.1.0 - A guided journey
This release focuses on structured learning in student-teacher setting.

### Features
- [X] Create/manage users 
- [x] Create/edit/delete learning journeys (teachers-only)
- [x] Create/edit/delete exercises in a learning journey (teachers-only)
- [x] Support for new content types, sentence, translation, conversation 
- [x] Support for viewing student progress (teachers-only)
- [x] Support for viewing student vocabulary (teachers-only)
- [x] Support for viewing student's answers (teachers-only)
- [x] Support for custom feedback to student's answers (teachers-only)
- [x] Support for language journey (incl. exercises, feedback, vocabulary tracking)
- [X] Dynamic sentence based exercises
    - arrange words in sentence exercise
    - fill the blanks exercise
- [x] Dynamic translation-based exercises 
    - select translate text exercise
    - fill-in blank translate text exercise 
    - write translate text exercise
    - pronounce translate text exercise
- [x] Dynamic conversation-based exercises
    - review conversation exercise
    - order conversation exercise
    - fill-in blank conversation exercise
    - transcribe conversation exercise
    - pronounce conversation exercise
