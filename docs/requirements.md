# Requirements 

## Exercises must be able to be associated with multiple sessions 
In order to implement the learning path feature, it must be possible to create exercises beforehand. This requires that exercises can be associated with multiple sessions.

## Exercise answers must be assocaited with a specific session
In order to track progress through a learning path, it must be possible to associate exercise answers with a specific session. This allows for accurate tracking of which exercises have been completed in which sessions.
